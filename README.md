# PythonTool

### CheckImageUser

功能：

1. 检索flutter工程下所有的png图片
2. 检查所有的png的图片有没有被使用
3. 删除未使用的png图片

### Tinypng

功能：

1. 检索flutter工程下目录下所有的png的图片

2. 对所有的png的图片进行压缩

3. 压缩后的图片替换原路径的图片

   

### check_andriodx

功能：

1. 检测Anroidx检测失败的问题



### statistics

功能：

1. 检测当前的汉字的个数



### transform_file

功能：

1. 将文件转换后转存到相应的目录
